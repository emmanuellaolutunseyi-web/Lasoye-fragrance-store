# Lasoye Perfume Store

A luxury-themed perfume e-commerce website built using HTML, CSS, and JavaScript.

## Features
- Responsive navbar 
- Hero section with branding
- Product carousel
- Seasonal collections
- Interactive hover effects
- Add-to-cart UI interactions

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Future Improvements
- Backend with Django or Flask
- User authentication
- Shopping cart system
- Payment integration
